# ZFORCE DMS – ROLE-BASED ACCESS & DATA CONTROL

## CORE ROLES (NON-NEGOTIABLE)

1. **HO Super Admin (ZForce HQ)**
2. **HO Department Admin** (Sales / Service / Finance)
3. **Dealer Principal**
4. **Dealer Sales Executive**
5. **Service Manager**
6. **Technician**
7. **CRM / Call Center**
8. **Finance Executive**
9. **Customer (Optional App)**

---

# 1. HO SUPER ADMIN (FULL CONTROL)

### Access

✔ ALL modules
✔ ALL dealers
✔ ALL historical data

### Can ADD / EDIT

* Dealers
* Users & Roles
* Pricing, Incentives
* SLA rules
* Warranty rules
* Vehicle & Parts Masters

### Pages Visible

* HO Dashboard (global)
* Sales / Service / Spares / Warranty (all dealers)
* Finance & MIS
* CRM Analytics
* Audit Logs
* System Configuration

### Cannot

❌ Perform dealer-level transactions (no bookings, no job cards)

# 11. DATA VISIBILITY RULES (CRITICAL)

* Dealer users **never see other dealers**
* Technicians **never see money**
* CRM users **never confirm sales**
* Finance users **never touch operations**
* Customers **see only their VIN**

Break these rules → data leaks → lawsuits.
