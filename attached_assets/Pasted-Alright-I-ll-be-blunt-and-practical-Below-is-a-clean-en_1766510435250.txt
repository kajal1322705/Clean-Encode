# ZFORCE DMS – Page Architecture & Reusable Component System

## 1. TECH STACK (SUGGESTED – NO FLUFF)

* **Frontend**: React + TypeScript
* **Routing**: React Router
* **UI**: Tailwind / MUI (pick one, don’t mix)
* **State**: Redux Toolkit / Zustand
* **Charts**: Recharts
* **Auth**: JWT + RBAC
* **API**: REST (later GraphQL if needed)
